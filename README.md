# AI Agent

An AI-powered conversational agent built using Python, FastAPI, LangChain, and OpenAI. This project demonstrates how to build an intelligent chatbot with a REST API backend and support for Large Language Models (LLMs).

## Features

- AI-powered chatbot
- FastAPI backend
- LangChain integration
- OpenAI API integration
- Modular and scalable project structure
- REST API endpoints

## Tech Stack

- Python
- FastAPI
- LangChain
- OpenAI API
- Git & GitHub
- VS Code

## Installation

### Clone the repository

```bash
git clone https://github.com/Adityahon3344/AI-Agent-.git
```

### Move into the project directory

```bash
cd AI-Agent-
```

### Install dependencies

```bash
pip install -r requirements.txt
```

## Running the Project

If using FastAPI:

```bash
uvicorn app.main:app --reload
```

Or run your Python file if applicable:

```bash
python main.py
```
